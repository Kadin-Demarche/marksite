# Welcome to Your Starter Blog

This starter comes with sample posts, tags, categories, and search enabled. Edit the content in `content/` and run `npm run build` to generate your site.

## What's Included

- Home, About, and Contact pages
- Sample posts with tags and categories
- RSS feed, sitemap, and dark mode
- Syntax highlighting and table of contents

## Next Steps

1. Update `config.yaml` with your site info.
2. Replace the sample posts in `content/posts/`.
3. Run `npm run build` and deploy to GitHub Pages.
