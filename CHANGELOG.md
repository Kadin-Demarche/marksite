# Changelog

## [1.0.0] - 2025-11-21

Initial release.

### Features
- Markdown to HTML conversion with frontmatter
- Blog system with pagination and tag pages
- Full-text client-side search
- Responsive theme with dark mode
- Syntax highlighting
- Automatic sitemap and RSS feed
- SEO optimization (meta tags, Open Graph, JSON-LD)
- CLI tool and dev server
- GitHub Actions workflow
